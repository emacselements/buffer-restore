# buffer-restore

Manual session management for Emacs. Save and restore named window layouts with precise buffer states, including PDF pages and EPUB positions.

## Installation

```elisp
(add-to-list 'load-path "/path/to/buffer-restore")
(require 'buffer-restore)
```

## Usage

**Sessions** (single frame):
- `M-x buffer-restore-save-session` - Save current layout
- `M-x buffer-restore-load-session` - Restore a saved layout
- `M-x buffer-restore-list-sessions` - View all sessions
- `M-x buffer-restore-delete-session` - Delete a session

**Workspaces** (all frames):
- `M-x buffer-restore-save-workspace` - Save all frames
- `M-x buffer-restore-load-workspace` - Restore all frames
- `M-x buffer-restore-list-workspaces` - View all workspaces
- `M-x buffer-restore-delete-workspace` - Delete a workspace

## What Gets Saved

- Window splits and sizes
- File buffers with cursor position
- PDF pages, zoom, and scroll (requires pdf-tools)
- EPUB chapters and position (requires nov.el)
- Frame positions and sizes (workspaces only)

## Optional Configuration

```elisp
;; Custom storage location
(setq buffer-restore-directory "~/my-sessions")

;; Keybindings
(global-set-key (kbd "C-c s s") 'buffer-restore-save-session)
(global-set-key (kbd "C-c s l") 'buffer-restore-load-session)
(global-set-key (kbd "C-c w s") 'buffer-restore-save-workspace)
(global-set-key (kbd "C-c w l") 'buffer-restore-load-workspace)
```

## Requirements

Emacs 26.1+
